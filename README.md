# Coffjoy ☕

Навчальний фронтенд‑проєкт, створений для практики сучасної розробки
веб‑інтерфейсів. Проєкт реалізує адаптивний лендинг кавового бренду на основі
готового дизайн‑макету.

## 📌 Мета проєкту

- Практика роботи з сучасним стеком фронтенд‑технологій
- Відпрацювання навичок верстки за макетом
- Робота з компонентною архітектурою
- Організація структури проєкту
- Дотримання best practices розробки

Проєкт виконується відповідно до чек‑листу із завданнями.

## 🛠 Технології

Проєкт побудований із використанням сучасного стеку:

- **React** — бібліотека для створення UI
- **TypeScript** — типізація JavaScript
- **Vite** — швидкий build tool та dev server
- **CSS / SCSS** — стилізація інтерфейсу
- **ESLint** — linting коду
- **Prettier** — форматування коду

## 📂 Структура проєкту

```
src/

components/      # reusable UI components
pages/           # сторінки додатку
styles/          # глобальні стилі
assets/          # зображення та іконки
hooks/           # custom React hooks
utils/           # helper functions

App.tsx          # root component
main.tsx         # application entry point
```

## ⚙️ Встановлення та запуск

### 1. Клонувати репозиторій

```bash
git clone https://github.com/your-username/coffjoy-project.git
```

### 2. Перейти у папку проєкту

```bash
cd coffjoy-project
```

### 3. Встановити залежності

```bash
npm install
```

### 4. Запустити dev сервер

```bash
npm run dev
```

Після запуску проєкт буде доступний за адресою:

```
http://localhost:5173
```

## 🧹 Якість коду

Для підтримки якості коду використовуються:

```bash
npm run lint
```

```bash
npm run format
```
